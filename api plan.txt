- 회원 가입//
     -Request
        -Method: POST
        -URL: /user
        -Body:{ email:string, name:String, age:Number, phone:Number }
    -Response
        -Status code:201
        -Payload: { token: string }
- 로그인//
    -Request
        -Method: POST
        -ULR: /login
        -Body:{ email:string, password:string }
    -Response
        -Status code:201
        -Payload: { token: string }
- 회원 정보 수정//
    -Request
        -Method: PUT
        -ULR: /user/:userId
        -Body:{ email:string, name:String, age:Number, phone:Number }
    -Response
        -Status code:200
        -Payload: { token: string }
- 회원 탈퇴//
    -Request
        -Method: DELETE
        -ULR: /user/:userId
    -Response
        -Status code:200
        -Payload: { message: string }
